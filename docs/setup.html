üöÄ Complete Setup Guide

Follow these steps to get your eBay AI Analyzer up and running in minutes!

Step 1: Clone the Repository

First, clone the repository to your local machine:

git clone https://github.com/Nick-Mccubbin00/EbaySearches.git cd EbaySearches
‚úÖ Repository cloned successfully!

Step 2: Install Python Dependencies

Install the required Python packages:

pip install -r requirements.txt

This will install:

  • requests: For making HTTP requests to eBay API
  • google-generativeai: For Google Gemini AI integration
  • typing-extensions: For type hints support
‚úÖ Dependencies installed successfully!

Step 3: Get Your API Keys

üîë eBay API Setup

  1. Go to eBay Developer Program
  2. Sign in with your eBay account
  3. Create a new application
  4. Get your OAuth access token
  5. Copy the token for the next step

ü§ñ Google Gemini API Setup

  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Create a new API key
  4. Copy the API key for the next step
‚ö†Ô∏è Important: Keep your API keys secure and never share them publicly!

Step 4: Configure API Keys

Open Complete_Ebay_AI_Analyzer.py and update the configuration section:

# eBay API Configuration EBAY_ACCESS_TOKEN = 'your-ebay-oauth-token-here' # Google Gemini API Configuration GEMINI_API_KEY = 'your-gemini-api-key-here'

Replace the placeholder values with your actual API keys.

‚úÖ API keys configured successfully!

Step 5: Test Your Setup

Run the analyzer to test your configuration:

The script will:

  • Search eBay for "2004 Silver Eagle MS69"
  • Apply AI confidence scoring
  • Generate a comprehensive report
  • Save results to JSON files
‚úÖ Setup complete! Your eBay AI Analyzer is ready to use.

‚öôÔ∏è Configuration Options

Customize the analysis parameters:

# Analysis parameters max_results = 20 # Maximum listings to analyze min_confidence = 70 # Minimum confidence score (0-100) days_back = 90 # Search period in days
  • max_results: Increase for more data, decrease for faster analysis
  • min_confidence: Higher values = more accurate but fewer results
  • days_back: How far back to search for sold items

üö® Troubleshooting

If you get "Invalid access token" errors:

  • eBay OAuth tokens expire regularly - refresh your token
  • Make sure you're using the correct token type (OAuth, not App ID)
  • Check that your eBay application is properly configured

If you get "No Gemini API key provided":

  • The system will fall back to rule-based scoring
  • This still works but without AI enhancement
  • Add your Gemini API key for full AI functionality
